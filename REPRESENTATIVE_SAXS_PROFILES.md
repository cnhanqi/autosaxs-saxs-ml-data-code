# Representative SAXS Profiles

The curated minimal public release retains the following ten representative SAXS profiles:

- `Buf_2.5mg.dat`
- `Buf_50mg.dat`
- `Buf_100mg.dat`
- `EAN1.5_50mg.dat`
- `EAN5.7_5mg.dat`
- `EAN12_50mg.dat`
- `EAN30_75mg.dat`
- `EAN54.6_25mgb.dat`
- `EAN75_2.5mg.dat`
- `EAN99_70mg.dat`

These files were chosen to span buffer control conditions and a broad range of EAN compositions and protein concentrations without redistributing the full profile collection.
