# Data and Code Availability

The data and code supporting this article are derived from SAXS measurements of lysozyme/EAN crystallization conditions and the associated descriptor-extraction and machine-learning workflow. The curated descriptor dataset (`n = 96`), ten representative SAXS profiles, lightweight Bragg-peak-detection and crystallinity-index helpers, a minimal OLIGOMER command wrapper, model-evaluation metric tables, the hyperparameter-grid description, and a written summary of random-seed, train/test split, and cross-validation settings are available at:

GitHub repository: https://github.com/cnhanqi/autosaxs-saxs-ml-data-code

The currently archived Zenodo release associated with this repository is:

Zenodo DOI: https://doi.org/10.5281/zenodo.20716687

The archived release tag currently cited for the study is:

Release tag: v1.0.2

ATSAS/OLIGOMER is third-party software and is not redistributed in this repository. Full raw SAXS working directories, figure-generation code, large OLIGOMER basis files, complete example-output bundles, and manuscript-preparation files are not included in this curated minimal public package.

No human participant data are included.
