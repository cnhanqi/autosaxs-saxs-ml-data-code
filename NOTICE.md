# Notice

This repository is a curated reproducibility package, not the full local working directory.

The following materials were intentionally excluded:

- manuscript drafts and reviewer-response working files
- PowerPoint, Word, Illustrator, Origin, compressed archive, and local IDE files
- unrelated exploratory analyses
- full raw beamline working folders not required for the reported descriptor/ML workflow
- third-party ATSAS executables

Legacy step scripts with encoding or syntax damage in the original working tree were moved to `legacy_reference_scripts/` as non-executable text files for provenance only. The recommended runnable implementation is in `src/` and `scripts/`.
